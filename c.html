OpenGL Scene

Draws a lit scene with various rotating objects. Camera can rotate with the mouse. The sphere is created by subdividing a tetrahedron.

OpenMP Laplace Heatmap

Simulates a heat distribution map with successive approximations computed in parallel.

OpenMPI Mandelbrot Set

Calculates and visualizes the Mandelbrot set in parallel with message passing.

OpenMPI N-Body Orbits

Simulates the planetary rotations of our system as an n-body problem, calculated in parallel with message passing. Pictured is a ten year orbit of the planets.
